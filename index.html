Some of Thing You can do in tecnology

Front-End

se refiere a la parte del desarrollo de software que se enfoca en la creación de la interfaz de usuario y la experiencia del usuario en una aplicación web. Los desarrolladores front-end trabajan con tecnologías que se ejecutan en el navegador del usuario.

Backend

se refiere a la parte del desarrollo de software que se enfoca en el servidor, las bases de datos y la lógica de la aplicación. Los desarrolladores back-end construyen y mantienen la tecnología que alimenta los componentes del lado del servidor.

Full-Stack

combina tanto el front-end como el back-end. Los desarrolladores full-stack tienen conocimientos y habilidades en ambas áreas y pueden manejar una aplicación completa desde la interfaz de usuario hasta la base de datos y el servidor.

Mobile Development

se refiere a la creación de aplicaciones para dispositivos móviles, como smartphones y tablets. Los desarrolladores móviles se especializan en construir aplicaciones que se ejecutan en sistemas operativos móviles como iOS y Android.

La tecnología tiene muchas rutas donde cada una de estas aparecen nuevas funcionalidades y metodologías para hacer las cosas.
